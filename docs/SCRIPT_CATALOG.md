## Detailed Script Information

### Dynamic Content Block Embedder
- **File**: apps/docs/src/docs-embed-content-block.gs
- **Purpose**: Automatically embed content from one Google Doc into another using placeholders
- **Features**:
  - Extracts content from a source document based on a header
  - Dynamically finds the end of the relevant content by looking for the next header or the end of the document
  - Replaces a placeholder in a destination document with the extracted content
- **Services**: Google Apps Script (DocumentApp)

### Google Docs Comments to Sheets Exporter
- **File**: apps/docs/src/docs-export-comments-sheets.gs
- **Purpose**: Extract all comments and replies from Google Docs and export to spreadsheet
- **Features**:
  - Extracts all comments and replies from specified Google Document
  - Processes comment metadata (author, date, status, content)
  - Calculates word count and character count for each comment
- **Services**: Google Drive API, Google Sheets API, Google Apps Script

### Google Docs File Inventory to Sheets
- **File**: apps/docs/src/docs-export-file-list-to-sheets.gs
- **Purpose**: Generate comprehensive file inventory with metadata for Google Docs in a folder
- **Features**:
  - Scans specified folder for all Google Docs files
  - Extracts file names, URLs, and direct links
  - Calculates word count and character count for each document
- **Services**: Google Drive API, Google Docs API, Google Sheets API

### Google Docs to Obsidian Markdown Exporter
- **File**: apps/docs/src/docs-export-markdown-obsidian.gs
- **Purpose**: Convert Google Docs to Obsidian-compatible Markdown with backlink generation
- **Features**:
  - Converts Google Docs to Markdown format
  - Extracts and lists all links from the document
  - Creates a Markdown file in a specified Obsidian folder
- **Services**: - Google Drive API (DriveApp)

### Batch Google Docs Formatter
- **File**: apps/docs/src/docs-formatter-content.gs
- **Purpose**: Apply consistent formatting styles across multiple Google Docs in a folder
- **Features**:
  - Batch processes all Google Docs in a specified folder
  - Applies consistent font family (Helvetica Neue) across all heading levels
  - Sets appropriate font sizes for title, subtitle, and H1-H6 headings
- **Services**: Google Drive API, Google Docs API
