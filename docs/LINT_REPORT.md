## Lint Rules

1. **trailing-whitespace**: No trailing whitespace at end of lines
2. **line-too-long**: Lines should not exceed 120 characters
3. **todo-comment**: TODO/FIXME comments should be resolved
4. **console-log**: Use Logger.log instead of console.log in GAS
5. **missing-semicolon**: Statements should end with semicolons
6. **missing-header**: Files should have header comments
7. **empty-file**: Files should not be empty
8. **excessive-blank-lines**: No more than 2 consecutive blank lines
